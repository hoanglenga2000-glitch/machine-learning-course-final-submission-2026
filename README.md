# 机器学习课程最终提交材料

本仓库为机器学习课程最终提交专用公开仓库，供老师审核课程作业使用。

GitHub 公开地址：
https://github.com/hoanglenga2000-glitch/machine-learning-course-final-submission-2026

## 提交内容

- `01_数据`：课程案例使用的数据、处理后数据、数据来源与清洗说明。
- `02_源码`：数据获取、清洗、建模、报告生成和核验相关代码。
- `03_小组汇报PPT和报告`：小组案例汇报 PPT、PDF 报告及展示材料。
- `04_小组成员个人作业整理`：小组成员平时作业汇总 PDF、10 次作业 PDF、个人案例总结。
- `15周最终提交要求核对.md`、`提交说明与质量核验.md`、`最终上交审核报告.md`：提交要求与文件完整性核验说明。

## 对应课程要求

1. 平时作业汇总、案例汇报、个人案例总结已分别整理。
2. 小组案例数据、代码、PPT 和 15 页以上 PDF 文档已放入对应目录。
3. 每位小组成员的平时作业汇总 PDF、个人 10 次作业 PDF、个人总结报告已按成员目录归档。
